# Focal-LightGBM-MoMo-Fraud-Ghana

**An Engineered Focal-LightGBM Model with SHAP Explainability for Real-Time Mobile Money Fraud Detection in Ghana**

## Overview
This repository contains the code, notebooks, and artifacts for the research proposal:  
**"An Engineered Focal-LightGBM Model with SHAP Explainability for Real-Time Mobile Money Fraud Detection in Ghana: A Hybrid Public-Field Transaction Dataset Approach"**

**Key Contributions**:
- Fabrication of focal loss into LightGBM (**Focal-LightGBM**)
- Integration of SHAP explainability
- Hybrid dataset (PaySim public + planned Ghanaian institutional logs)
- Reproducible baseline vs. engineered comparison
